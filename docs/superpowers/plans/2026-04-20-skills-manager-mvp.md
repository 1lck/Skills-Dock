# Skills Dock MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first working Skills Dock desktop MVP on Tauri with local multi-source skill discovery, search, validation, and detail preview.

**Architecture:** Use a Tauri shell with a React and TypeScript frontend plus a Rust command layer. Keep scanning and normalization logic behind a small application/domain layer so the UI only consumes unified source and skill models.

**Tech Stack:** Tauri, React, TypeScript, Vite, Rust, Vitest, Testing Library
